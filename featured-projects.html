Movie Recommendation System using Sentiment Analysis

Languages & Tools: Python, Pandas, NLTK, Scikit-learn

Algorithms: TF-IDF Vectorization, Naive Bayes Classification

Goal: To recommend movies by analyzing user reviews for sentiment.

Steps Followed:

  • Collected and cleaned movie review data using Pandas.
  • Applied NLTK for sentiment labeling and tokenization.
  • Used TF-IDF to convert text into numerical features.
  • Trained a Naive Bayes classifier to detect sentiment trends.
  • Integrated sentiment scores to recommend positively-rated movies.

Outcome: Built a sentiment-aware recommendation engine that suggests movies based on emotional tone.

KNN vs Scikit-learn Performance Comparison

Languages & Tools: Python, NumPy, Scikit-learn, Matplotlib

Algorithms: K-Nearest Neighbors (custom + Scikit-learn)

Goal: To compare a manual implementation of KNN with Scikit-learn’s version for accuracy and speed.

Steps Followed:

  • Implemented KNN manually using NumPy distance calculations.
  • Loaded datasets and ran both manual and sklearn models on them.
  • Measured performance using accuracy scores and runtime benchmarks.
  • Visualized decision boundaries using Matplotlib.

Outcome: Learned trade-offs between custom algorithms and optimized library solutions.
